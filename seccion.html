Como agregar stilos CSS a una pagina html

HTML
	
                	
<!DOCTYPE html>
<html>
	<head>
		<title>Hola Mundo</title>
		<!--Esta linea hace referencia nuesro documento css-->
		<link href="css/MyStile.css" rel="stylesheet">
	</head>
	<body>
		<div>
			Hola soy un div
		</div>
	</body>
</html>

CSS
	
                	
.MyStyle1{
	color: black;
	display: flex;
	position: absolute;
}
